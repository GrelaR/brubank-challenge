## Getting Started

### Prerequisites

* You should have android studio installed on your computer and an emulator device set up.

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:GrelaR/BankuishChallenge.git
   ```
2. Open it on android studio
3. Run it on the device you have setup.

<!-- USAGE EXAMPLES -->
## Usage

The project is about retrieving movies from https://developer.themoviedb.org/docs/getting-started and displaying it on a list, each item will open a new screen with details of the item clicked. The user can suscribed to the movie, and also can search for movies introducing a query parameter with the movie's name. Details designs should be improved and more tests are needed :)
